# 👩🏻‍🎓NCYU CSIE 招生貼文

* 主題是為嘉大資工系做一個招生導向的 APP
* 由於客群是剛準備升大學的高中生們，因此希望**以有趣、活潑且新穎的方式來介紹我的校園生活**
* 透過類似社群軟體的介面來呈現一篇招生貼文

## 實際介面
* 點籍「偷偷查看更多...」展開文字查看全文，注意文字內容有一些小彩蛋🥳
